# AgentOStudio Product Demos

Showroom público e interativo dos produtos privados AgentOStudio.

## Segurança e separação

Este repositório contém somente uma simulação independente com dados fictícios. Não inclui código, banco de dados, credenciais, integrações, regras internas nem arquivos dos produtos privados.

## Demonstração escolar

Use o botão **Explorar sistema escolar**, entre com os dados já preenchidos e navegue pelos módulos simulados. Nenhuma informação é enviada ou armazenada.
